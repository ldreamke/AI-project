# 股票组可行性报告

## 一、问题定义

​    本项目将展示如何使用深度神经网络来预测股价，属于回归问题。本项目的核心步骤包括：

1. 从Yahoo财经下载股票数据；
2. 根据神经网络库的数据格式要求对数据进行预处理；
3. 训练神经网络模型；
4. 用历史数据进行回测。

## 二、数据

​    本项目所用数据来自雅虎财经。
